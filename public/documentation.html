myFlix API Documentation

This page will provide users with access to information about different actors, directors, genres, and movies(rating and release date). Also users will be able to sign up, update their personal information, and create a list of their favorite movies.

Below we introduce some key concepts, further explore endpoints, and then present example use cases that map generally to a specific endpoint. Hopefully, this information will help you more efficiently integratee all endpoints, or help you decide which endpoint best fits your mission.

Request URL Endpoints HTTP Methods Request Body Response Body
Get a list of movies. /movies GET None A JSON object holding data about all the movies.
Get data about a single movie by title. /movies/:Title GET None A JSON object holding data about a single title, containing an ID, description, genre, director, and image URL.
{
"_id": "",
"Title": "",
"Description":"",
"Genre":{ }
  "Name": "",
  "Description": ""
},
"Director":{
  "Name": "",
  "Bio": "",
  "Birth": ""
},
"Actors": [],
"ImagePath": ""
"Fetured": Boolean
}
Get data about a genre by name title. /genre]/:Name GET None A JSON object holding data about a single genre, containing description. Example:
{
description: string
}
Get data about a director by name. /director/:Name GET None A JSON object holding data about a single director, containing name, bio, birth year, and death year. Example:
{
Director: {
  name: string,
  bio: string,
  birth year: number,
  death year: number
  }
}
Add a user. /users POST A JSON object holding data to update with new user name. Example
{
Username: string,
Password: string,
Email: strong,
favoriteMovies: string
} 		A JSON object holding the updated user name and given id. Example:
{
id: number,
Username: string,
Password: string,
Email: string,
favoriteMovies: string
}
Update username. /users/:Username PUT None A text message indicating the username that was updated.
Add a movie. /users/:Username/movies/:[id] POST A JSON object holding data, new favorite movie title. Example:
{
name: string,
favoriteMovies: string
} 		Send text message movie title has been added to user id's array. As JSON object shows:
{
id: number,
Username: string,
Password: string,
Email: string,
favoriteMovies: string
}
Delete a favorite movie title from user's list. /users/:Username/movies/:[id] DELETE None A text message indicating whether the favorite movie title was successfully removed.
Deregister user. /users/:Username DELETE None A text message indicating user has been successfully removed.
